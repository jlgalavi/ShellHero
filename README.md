Level up while you code! A virtual companion in your terminal that evolves as you work.

## 📖 About the Project

This is an Open Source, DIY project aimed at adding a layer of gamification to the Linux development environment. 
As you work, compile, or commit in your terminal, your companion will gain experience (XP), level up, and unlock achievements.

## ✨ Features (Planned)

- **Your Avatar:** An ASCII/Emoji character that greets you upon opening the terminal.
- **XP System:** Earn points for executed commands, session time, or development milestones.
- **Achievements:** Unlock visual badges by reaching specific goals.

## 🔒 Privacy by Design (Local-First)

The development environment is sacred. **This project runs 100% locally.** 
We do not collect command data, there is no telemetry, and no external servers are involved. All progress is saved in a local configuration file on your machine.
